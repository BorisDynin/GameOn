# Svelte 5 Migration - GameON

This is a modernized version of the GameON project, completely refactored and migrated to Svelte 5.

## Installation

1.  Navigate to the `svelte-v5-migration` directory:
    ```bash
    cd svelte-v5-migration
    ```
2.  Install the dependencies:
    ```bash
    npm install
    ```

## Running the Project

1.  To start the development server, run:
    ```bash
    npm run dev
    ```
2.  Alternatively, you can use the provided `start.sh` script to install dependencies and start the server in one command:
    ```bash
    ./start.sh
    ```

The application will be available at `http://localhost:5173`.
